## вҡҷпёҸ Settings

| Setting | Default | Options |
|:--------|:-------:|:--------|
| `printCode.lineNumbers` | `true` | Show / hide line numbers |
| `printCode.header` | `true` | Show / hide file header (name, path, date) |
| `printCode.colorMode` | `"colored"` | `"colored"` вҖ” syntax colors В· `"black"` вҖ” monochrome |

> рҹ’Ў **Tip:** Access settings quickly by hovering over the status bar icon and clicking **Settings**.
